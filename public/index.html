Short link microservice

A freeCodeCamp project by Ilya Murashov

This service allows simple conversion between UNIX and human readable time formats.

Usage example

https://[service_address]/December 15, 2015
or
https://[service_address]/1450134000000

Response example

In both query scenarios you should get the output in the following format:

{
    "unix": 1450134000000,
    "natural": "Tue Dec 15 2015"
}
