Short link microservice

A freeCodeCamp project by Ilya Murashov

This service allows URL shortening.

Usage example

To add a new link type in the desired URL after /new/ (please omit http[s]:// from the URL):

https://fcc-im-shortly.herokuapp.com/new/facebook.com
or
https://fcc-im-shortly.herokuapp.com/new/www.facebook.com

You will be notified about the shortened version of the link in the format which is similar to
{"full-url":"facebook.com","short_url":"fcc-im-shortly.herokuapp.com/16"}
In this example, to be redirected to facebook.com follow the fcc-im-shortly.herokuapp.com/16 link.
